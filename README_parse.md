# Парсер цен на диваны с divan.ru

## Описание
Скрипт для парсинга цен на диваны с сайта https://www.divan.ru/category/divany с использованием библиотеки Selenium.

## Установка зависимостей

1. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

2. Установите ChromeDriver:
   - Скачайте ChromeDriver с https://chromedriver.chromium.org/
   - Или установите через менеджер пакетов:
     - macOS: `brew install chromedriver`
     - Linux: `sudo apt-get install chromium-chromedriver`
     - Windows: скачайте и добавьте в PATH

## Использование

Запустите скрипт:
```bash
python parse_divan.py
```

Результаты будут сохранены в файл `divan_prices.csv`.

## Структура CSV файла

Файл содержит следующие колонки:
- **Название** - название дивана
- **Цена** - цена в рублях (только цифры)

## Примечания

- Скрипт работает в headless режиме (без открытия окна браузера)
- Если парсинг не работает, попробуйте убрать `--headless` из опций Chrome для отладки
- Скрипт ограничен парсингом первых 50 товаров для примера (можно изменить в коде)
